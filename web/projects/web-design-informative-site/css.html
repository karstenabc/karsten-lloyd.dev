CSS

Introduction to CSS

CSS stands for Cascading Style Sheet. This is what determines how the content from the HTML document is displayed in the browser. CSS can also be used to change the appearance of the individual elements such as text and images.

Stylesheets are made up from a combination of individual style properties which will be applied to one or more HTML element.


Addressing HTML Elements

There are 3 ways to select HTML elements in a document. Click on each title to view the content.

1. By using IDs

IDs should be unique and will therefor identify a single element in the document. They are assigned in the opening tag in the HTML.

Code

CSS in HTML head tag

<head>
    <style>
        #paragraph_1 {
            color: red
        }
    </style>
</head>

HTML

<body>
    <p id="paragraph_1">paragraph 1</p>
    <p>paragraph 2</p>
    <p>paragraph 3</p>
</body>

Browser Output

paragraph 1

paragraph 2

paragraph 3

As you can see, the first paragraph is the only one that has had the color property applied to it.

2. By using classes

Classes can be added to any number of HTML elements. This can be achieved in two ways:

Either by adding the class to the elements in their opening tag like adding an ID or by using a div (see below).

Code

CSS in HTML head tag

<head>
    <style>
        .paragraph_style {
            color: red
        }
    </style>
</head>

HTML

<body>
    <p class="paragraph_style"> paragraph 1</p>
    <p class="paragraph_style"> paragraph 2</p>
    <p> paragraph 3</p>
</body>

Browser Output

paragraph 1

paragraph 2

paragraph 3

Here, the paragraphs that have been assigned the paragraph_style class have been styled in identicle ways. The element without a class assigned to it has not been styled.

3. By using a div

A div is like a container which can hold multiple HTML elements and can be used to position or style such elements in the same way. The styles work by addressing the div, by class or ID and then selecting the desired tag types within the div to style.

For this example, we will be styling all of the p tags, so this is what will be selected.

Code

CSS in HTML head tag

<head>
    <style>
        .paragraph_container p {
            color: red
        }
    </style>
</head>

HTML

<body>
    <div class="paragraph_container">
        <p>paragraph 1</p>
        <p>paragraph 2</p>
        <p>paragraph 3</p>
    </div>
</body>

Browser Output

paragraph 1

paragraph 2

paragraph 3

Here, none of the paragraph elements have been assigned an ID, but they have all inherited the style from the div element.



Implementing Styles into HTML

There are 3 ways to insert styles into a HTML document:

1. Inline with the HTML element

Inline styles will only apply to the element that it is inserted into. Multiple styles can be applied and are seperated by a semicolon.

Code

Inline CSS

<p>paragraph 1</p>
<p style="color:red;">paragraph 2</p>
<p>paragraph 3</p>

Browser Output

paragraph 1

paragraph 2

paragraph 3

2. Inserting into the head tag of the HTML document

Styles can be added into the head tag of the HTML document. This is achieved by inserting the styles into a style tag.

Code

HTML Head

<head>
    <style>
        .p_style {
            color: red
        }
    </style>
</head>

HTML Body

<body>
    <p> paragraph 1</p>
    <p class="p_style"> paragraph 2</p>
    <p class="p_style"> paragraph 3</p>
</body>

Browser Output

paragraph 1

paragraph 2

paragraph 3

3. Inserting a stylesheet (recommend)

Styles can be written in an independent file called a stylesheet. This is considered best practice as it allows the styles to be changed without needing to modify the HTML document.

This method also saves time if the sheet is imported into multiple pages of a website. As any changes to the styles will apply to the whole site. Importing multiple sheets can be used to apply basic styles to a page other sheets imported after this will apply more page specific styles.

The last stylesheet to be imported has its styles take precedence over those prior to it.

Code

HTML Head

<head>
    <link rel="stylesheet" type="text/css" href="stylesheet.css">
    </style>
</head>

HTML Body

<body>
    <p> paragraph 1</p>
    <p class="p_style"> paragraph 2</p>
    <p class="p_style"> paragraph 3</p>
</body>

Browser Output

paragraph 1

paragraph 2

paragraph 3

This will work exactly as before, wehreby the classes can be accessed as if they were written in the head tag.

CSS - Text Styles

Introduction to CSS text styles

CSS can be used to modify the appearance of text and other elements on a page. This makes the page look more attractive.

Text Color

The color property will change the text colour of the selected elements. This can set either by using one of 140 colour names which are supported by all modern browsers eg: CadetBlue. For the full list, click here.

Alternatively HEX codes can replace the colour name. This will allow up to 16 million colours to be chosen.

An exmple of setting the same text color style:

<p style="color:cadetblue;">Text</p>
<p style="color:#5F9EA0;">Text</p>

Text Alignment

The text-align property can help with the appearance of the site by changing where the text will be displayed on each line. This can make elements stand out, such as: having titles centered and the main body being aligned left.

There are 4 different alignment options in CSS:

  1. Left
  2. Right
  3. Center
  4. Justify

<p style="text-align:left;">Text</p>

Text Font

The font-family property will determine the font of the text. This is achieved with a font stack. This is where multiple values are provided to this property sperated by commas. This will allow for fallback fonts which will be used if the browser does not support the first font specified.

This property can take either:

Family name
The name of a font-family, like "times", "courier", "arial", etc.
Generic Family
The name of a generic-family, like "serif", "sans-serif", "cursive", "fantasy", "monospace".

Start with the font you want, and always end with a generic family, to let the browser pick a similar font in the generic family, if no other fonts are available. Note: Separate each value with a comma. Note: If a font name contains white-space, it must be quoted. Text Style

The font-style property changes how the text is rendered on a page, similar to the HTML style tags. These different styles can make different pieces of text stand out.

There are 5 different alignment options in CSS:

  1. Normal
  2. Italic
  3. Oblique
  4. Inherit - The style set by its parent
  5. Initial - The default style

<p style="font-style:italic;">Text</p>

A weight of 400 is equal to normal and 700 is equal to bolde

An exmple of setting the same font weight:

<p style="font-weight: bold;">Text</p>

CSS - Layout Styles

Introduction to Position Properties

Any element has four layout properties:

Size -
The height and width of the element.
Padding -
The space between the conent of the element and its border (its size).
Border -
The outline of the element.
Margin -
The space between the border of the element and the surrounding content.

Layout Properties

Margin

Padding

Size

To asign values to these properties, use the following:
To create a 10px gap from all sides between the border and content of the selected element: padding = 10px;
To create a 10px gap from the left side between the border and the content of the selected element: padding-left: 10px;

There are four values for each of these properties:

  1. left
  2. right
  3. top
  4. bottom

Introduction to Div Elements

A div element is useful for grouping multiple HTML elements together. They can also be used to house background images. So, how can this container be aligned?

The default location for a div, is in the upper left corner of its container (eg: the document).

An exmple of creating a div:

<div>
    <p>My Div</p>
</div>

Browser Output

My Div

Aligning a Div (Margins)

A div element can be aligned relative to its parent (surrounding container).

They can be aligned horizontally using their margin property.

An exmple of aligning to the right:

.center_div {
    margin: 0 auto;
}

Setting margin to auto, means setting it to the max value. As neither, margin-left or margin-right has been set, then both will take equal values. This will center the element.


.right_div {
    margin-left: auto;
    margin-right: 0;
}

Browser Output

Center

Right

The margin property can take any value, usually in pixels px. These examples set the margins to 0. This is used to either reset the margin of an element or to positon an element exactly to the specified side.

Aligning Div Contents (Padding)

An elements contents is aligned to the top right by default, this can be adjusted by changing the padding property.

An exmple of padded content:

.text_div {
    margin-left: 10px;
    margin-top: 20px;
}

The padding property can take any value, usually in pixels, px. This will move the contents of the container accordingly.

Element Borders

The border property defines how the outline of an element is drawn. The default value varies for the type of element.

This property has a few attributes:

Color -
The border colour, can be HEX or CSS avaliable colour.
Width -
The border thickness, can be thin, medium, thick or any px value.
Style -
There are 10 border styles available:
  1. solid
  2. dashed
  3. dotted
  4. double
  5. ridge
  6. groove
  7. inset
  8. outset
  9. hidden
  10. none
Spacing -
The space between the border of the element and the surrounding content.
Radius -
This makes the corners rounded. A value of 50% will make the border a circle or oval depending on the width and height.

Border example:

.border_div {
    border-color: red;
    border-width: 2px;
    border-radius: 10px;
}

The padding property can take any value, usually in pixels, px. This will move the contents of the container accordingly.

Element Size - Width

An elements horizontal size can be changed by modifying its width property. The default is to stretch to fit its contents (eg: an image or text).

An exmple setting a different width:

.text_div_width1 {
    width: 50%;
}
.text_div_width2 {
    width: 50px;
}

The width property can take a percentage of its parent or any pixel, px. If the container is too thin for the content, then the text will wrap to another line.
