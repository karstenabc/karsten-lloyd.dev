HTML

Introduction to HTML

HTML stands for HyperText Markup Language. This is what tells the computer what information to put in the page. It can also provide some basic styles for the text.

HTML tags are used by the browser to determine how to interpret the contents of the tags.

A HTML element consists of three parts; an opening tag, the content, and a closing tag. The type of element, determined by the tag, is written inside of < and > brackets. For example, a paragraph tag would look like the following: <p>My paragraph.</p>, where the 'p' stands for paragraph. Every tag must have a matching closing tag so that the browser can interpet it correctly.

Anchor Tags

Anchor tags are used to create hyperlinks. These tags can be used to make any HTML element link to another piece of text or website.
Browsers often display this as blue, unerlined text (example) which will go purple when you have visited the link.
The code for an anchor tag is: <a href="linkhere">Text to display</a>. An optional additional attribute is download. This will download the contents of a link instead of opening it in the browser.

List Tags

Lists can be used to neatly display information, whether or not it needs ordering. For example, lists can provide simple navigation for websites and pages. There are 3 types of HTML list, each displaying the content slightly differently to the rest.

Unordered List

An unordered list is a stanadard bullet point list.

An exmple of what this list looks like:

  • Unordered item 1
  • Unordered item 2
  • Unordered item 3

Ordered List

An ordered list is similar to an ordered list but the items are numbered.

An exmple of what this list looks like:

  1. Ordered item 1
  2. Ordered item 2
  3. Ordered item 3

Description List

Description lists are used to show elements and have more detail below them.

An exmple of what this list looks like:

Term 1
Description 1
Term 2
Decription

Nested Lists

Nested lists allows multiple styles of lists to be used within eachother

Example: An unordered list within an ordered list

  1. Item 1
    • Item 2a
    • Item 2b
  3. Item 3

HTML - Object Tags

Introduction to HTML Object Tags

HTML objects allow multimedia elements to be inserted into web pages. These include images, videos, tables and more.

Image Objects

Image tags <img> can be used to insert images and animated images (GIFs) into a HTML document.

These are self closing tags which means the ending tag is not </img> but instead />

For example, the image on the right is inserted using the following HTML:
<img>src="images/html5.png alt='HTML5 Logo'"/>

If the image doesn't load, the alt text will be used -
Click to toggle image load

HTML5 Logo

The img tag has a few attributes:

src: This is where the image is located, this can be a URL or a local file.
alt: This is a description of the image and will be the text that will be displayed if the image doesn't load for any reason. This is also used for screenreaders, so they know what is in the image.
title: This is the text that is displayed if the user hovers over the image.
example:
<img>src="directory/filename.extension" alt="image description" title="image title for mouseover" />

HTML - Text Style Tags

Introduction to HTML Text Stlye Tags

HTML5 allows for a variety of text styles that can be applied to the various text tags. These can be used in addition to the headings to bring attention to different pieces of text.


Style Name Style Tag Example

Bold or Strong

<b> or <strong>

 This will make the text appear bold.

Italic or Emphasised

<i> or <em>

 This will make the text appear in italics but <em> is used in screenreaders to 'find' important information on the page.

Underline

<u>

 This will make the text appear underlined.

Highlight

<mark>

 This will make the text appear highlighted or 'marked'.

Small

<small>

 This will make the text appear smaller.

Insert

<ins>

 This will make the text appear underlined but is less more useful after a <del> tag.

Delete

<del>

 This will make the text appear with a strikethrough.

Superscript

<sup>

 This will make the text appear above other text.

Subscript

<sub>

 This will make the text appear below other text.
